## React Boilerplate

---

### Steps to start the application

1. Install the app dependencies

```
npm install
```

2. Start the application

```
npm start
```

3. Build the application (Production Purpose)

```
npm run build
```

### Features

- Babel 7
- WebPack 4
- Eslint
- Prettier
- Production Build
